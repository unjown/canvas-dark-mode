Why does this plugin disable itself when it updates?! It's very annoying!

This is a restriction with how chrome plugins deal with security on the URLs they work on. When I add your school's URL to this plugin's list, it will trigger an update in which you will have to approve the plugin to work on the new URls. As a developer, I cannot work around this other than with brute force of adding urls. Chrome does not let me use wildcards in the domain of the URL. So "canvas.*.edu" is not allowed. Though if it was, we would not have this issue.
